O código foi implementado usando a linguagem Java, agora funcionando perfeitamente usando sockets, então foi desenvolvido um algoritmo recursivo que busca links dentro do html do endereço, imprimindo na tela e já procurando mais links dentro deste.

O repositório git encontra-se no endereço https://github.com/tamarameneghetti/Trabalho-Redes

Para a execução do código, por linha de comando, deve-se digitar "make run", para editar a profundidade e a url deve-se editar no makefile

exemplo:

	http://windowsphonebrasil.com.br/ 
	profundidade 1
